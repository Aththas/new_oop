# MEI(Mercy Educational Institute) Admin Panel (Website)
Dive into the world of knowledge with the Mercy Educational Institute's E-Library. Access a vast digital repository of educational resources, books, and research materials. Empower your learning journey through seamless navigation, online borrowing, and a rich collection that nurtures intellectual growth.

## Installation
To run MEI Admin Panel Website locally, follow these steps:

1. Install Xampp server (or wamp server)
2. Clone the repository into C:\xampp\htdocs folder: git clone https://github.com/Aththas/new_oop.git
3. Start apache through xampp server
4. Configure firebase database
5. Access the website through localhost

## Usage
Once you have MEI Admin Panel Website up and running, you can perform the following actions:

### Admin
1. Login and manage admin profile.
2. Password reset through OTP (Email Verification).
3. Add and modify books.
4. View feedbacks and ratings of each book.
5. Upload resources to the firestore cloud storage.
6. Create staff and student accounts.
7. View tasks and the submissions of each task.
8. View the Innovations, accept/reject pending innovations. (After this action, system automatically sends an email to notify the users saying that their innovation has been accepted/rejected)
9. Create and manage resources such as news papers, past papers and practicles.
10. View students, books and staff details and download them as an excel sheet.

## Technologies Used
MEI Admin Panel Website is built using the following technologies:

1. Frontend: Html, css, js
2. Backend: PHP
3. Database: Firebase
4. Additional: PHP Mailer Function
